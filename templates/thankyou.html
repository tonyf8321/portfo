Password Checker

This project was made in order to allow someone to check whether a password has been hacked or not through using the first five character of the password's Sha1 hash in order to determine. Using the first five character of the hash ensures that if someone were to use this password for their account, they can do so knowing that only the first five hashed characters were sent through the internet instead the full forty characters of the hash.

News Scraper

This program was made to save time from having to sift through news sites by gathering only the top stories and displaying them back on the screen with a link followed by the title. It can be tedious going through countless stories only just to get the top ones out there. That is why this program was built. It is conveinent and saves a lot of time through a click of a button. The news articles are obtained through hacker news, which an extremely popular news site.

Image Processor

This works great with any type of photos, whether its a photo of nature or family photos that someone may want to improve the quality of to then frame the picture, which is another great use for this program. Also, it can improve the quality of cartoon like or game like photos drastically.
